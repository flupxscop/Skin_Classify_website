# 🤖 Skin Disease Classification Web

A web application for skin disease classification using AI (CNN + Metadata).
Built with Next.js and integrated with machine learning models.

---

## 🚀 Features

* 🧠 AI-based skin disease prediction
* 📸 Image upload for classification
* 📊 Combine image + metadata analysis
* ⚡ Fast and responsive UI

---

## 🛠 Tech Stack

* Next.js (React)
* TypeScript
* Tailwind CSS (if used)
* Python (for AI model)
* CNN + Naive Bayes

## 🚀 Getting Started

### 1. Clone repo

```bash
git clone https://github.com/flupxscop/Skin_Classify_website.git
cd Skin_Classify_website
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run project

```bash
npm run dev
```

### 4. Open browser

```
http://localhost:3000
```
